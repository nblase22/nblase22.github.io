Surf's Up Weather Analysis

The purpose of this project was to analyze a weather data set and use the results to help plan a hypothetical trip to Hawaii. This was done by plotting temperature and precipitation data over a given time period.

After cleaning the dataset, SQLAlchemy was used to create an sqlite database. This database was then used for climate analysis and exploration. A flask app was then created to run the python code.
